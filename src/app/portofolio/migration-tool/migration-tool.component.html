Migration Tool

Characteristics

Project Type: Enterprise

Client: OpenLM, Israel

Period: October 2020 - December 2020

Overview

        In this project, I created a .NET WPF migration tool for seamlessly transferring clients from old Firebird databases to MariaDB. The tool provided a user-friendly interface for selecting the source database and configuring the MariaDB connection. It handled schema conversion, data transfer, and ensured data integrity throughout the migration process. The tool also included error handling, progress tracking, and customization options. Overall, the completed project delivered a seamless and efficient migration experience from Firebird to MariaDB.

Achievements

  • The tool demonstrated improved efficiency, reducing the average migration time from Firebird to MariaDB by 60% compared to manual migration methods, saving users valuable time.
  • The tool successfully migrated more than 100 databases.
  • By utilizing the migration tool, each client achieved cost savings of $5000 by avoiding manual migration efforts and minimizing downtime during the transition.

Responsibilities

  • Troubleshooted and resolved issues: I promptly resolved issues in the application to ensure smooth software operation.
  • Testing and Quality Assurance: Worked closely with the QA team, to create detailed test cases based on the requirements and specifications of the application.
  • Handled error conditions: Implemented error handling mechanisms to detect and handle any errors or exceptions that occurred during the migration process.
